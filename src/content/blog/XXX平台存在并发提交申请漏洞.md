# **XXX平台存在并发提交申请漏洞**

**1.漏洞详情**

**在对XXX进行安全测试中，发现其存在并发提交审核漏洞，突破一个账号只能提交一条和一条手机验证码只能交一条的限制，能给工作人员造成严重工作负担**



**2.漏洞证明**

**提交申请页面抓包**

**![img](https://cdn.nlark.com/yuque/0/2026/png/60657001/1779785701629-9032ba0d-6792-44b2-ba90-66db40eb6a8b.png)**

**并发**

**![img](https://cdn.nlark.com/yuque/0/2026/png/60657001/1779785726655-cf0387c3-b230-4a5e-8aa4-6379f253b400.png)**

**突破一个账号只能提交一条和一条手机验证码只能交一条的限制**



**3.修复方案**

**用工单 ID + 用户 ID 生成 key，加 5 秒缓存锁（Redis / 内存都行）同一工单同一人，5 秒内只允许一次操作**
