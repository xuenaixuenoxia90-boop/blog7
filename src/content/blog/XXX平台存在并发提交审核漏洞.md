# **XXX平台存在并发提交审核漏洞**

**1.漏洞详情**

**3.修复方案**

**用工单 ID + 用户 ID 生成 key，加 5 秒缓存锁（Redis / 内存都行）同一工单同一人，5 秒内只允许一次操作**
