# **XXX存在并发破限查看文章漏洞**

**1.漏洞详情**

**在对进行安全测试时，发现其查看 VIP 文章接口存在并发漏洞，通过该漏洞查看的文章远超限制，造成财产损失**



**2.漏洞证明**

**随便点击一篇文章**

**![img](https://cdn.nlark.com/yuque/0/2026/png/60657001/1779787107772-3c4ca39f-e603-4226-8678-103397393c47.png)**

**发现会员限制，注意 url 中的文章 id**

**![img](https://cdn.nlark.com/yuque/0/2026/png/60657001/1779787126214-ffd7f86d-d57a-4a36-9971-39e9fff5c800.png)**

**发现查看会员文章的数据包中 id 就在 url 中**

**![img](https://cdn.nlark.com/yuque/0/2026/png/60657001/1779787147424-1dac25a5-9a03-430e-af37-57f0e73937e4.png)**

**复制 20 个文章的 id，在我已经使用了 2 个解锁文章的情况下，在重放器里面并发能并发 8 个左右**

**![img](https://cdn.nlark.com/yuque/0/2026/png/60657001/1779787186184-e9ec7741-3d48-4441-837c-190e0469c0bb.png)**

**远超非会员只能使用 5 个的限制**
