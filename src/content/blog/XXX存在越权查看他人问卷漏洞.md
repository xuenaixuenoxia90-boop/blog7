# **XXX存在越权查看他人问卷漏洞**

**1.漏洞详情**

**在对XXX进行安全测试时，发现其平台存在越权漏洞，通过修改 id 的方式查看别人的创立问卷，并获取问卷的敏感信息。**



**2.漏洞证明**

**这个漏洞是在XXX APP 里面测试得到的接口，接口调用的是XX问卷的。**

**进入XXX APP ，点这一栏有一个企业申购之类的接口（我已经提交过了这个接口在前端就隐藏了）**

**点进来之后抓包**

**看到这里有一个查询问卷的数据包**

**看到这里的 id 随机跑了一下**
